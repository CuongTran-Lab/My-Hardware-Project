This is an ESP32-C5–based control board, built around Espressif’s latest ESP32-C5 chipset with 5 GHz Wi-Fi support.
The board features 4 digital inputs and 4 digital outputs, designed to operate with industrial voltage levels ranging from 12 V to 24 V.
In addition, it includes one dedicated signal input, UART communication support, and status LEDs for real-time system state indication and diagnostics.
